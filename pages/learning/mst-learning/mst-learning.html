Spanning Tree Fundamentals

A spanning tree of a connected graph is a subgraph that contains all vertices while maintaining connectivity without forming cycles.

Properties of a Spanning Tree

  • Contains all vertices of the graph
  • Contains exactly V − 1 edges
  • Contains no cycles
  • Graph remains connected
Example
Graph:
A-B
A-C
B-C
B-D

Possible Spanning Tree:
A-B
A-C
B-D
02

Minimum Spanning Tree (MST)

A Minimum Spanning Tree is a spanning tree whose total edge weight is minimum among all possible spanning trees.

MST Characteristics

  • Includes all vertices
  • Contains V−1 edges
  • No cycles
  • Minimum possible cost
      A
     / \
    2   4
   /     \
  B---3---C

MST:
A-B (2)
B-C (3)

Cost = 5

A-B = 4
A-C = 2
B-C = 5
B-D = 10
C-D = 3

MST:
A-C = 2
C-D = 3
A-B = 4

Total Cost = 9
03

MST Properties

  • MST contains exactly V−1 edges.
  • Removing any MST edge disconnects the graph.
  • Adding any extra edge creates a cycle.
  • A graph may have multiple MSTs with equal cost.
04

Prim's Algorithm

Prim's Algorithm grows the MST one vertex at a time.

Steps

  1. Start from any vertex.
  2. Select the minimum edge connecting MST to a new vertex.
  3. Add the edge.
  4. Repeat until all vertices are included.
Time Complexity: O(E log V)
Space Complexity: O(V) 

Start Vertex

while MST not complete:
   choose minimum edge
   connecting tree to new vertex
   add edge to MST
05

Kruskal's Algorithm

Kruskal's Algorithm builds MST by selecting the smallest edges first while avoiding cycles.

Time Complexity: O(E log E)
Space Complexity: O(V) 

Sort edges by weight

for each edge:
   if cycle not formed:
      add edge to MST

Union Find (DSU)

Disjoint Set Union helps detect cycles efficiently in Kruskal's Algorithm. 


find(x)

union(x,y)

if find(u) != find(v):
    union(u,v)

Practice Exercises

Exercise 1

How many edges are present in an MST with 10 vertices?

An MST always contains V − 1 edges.

For 10 vertices: 10 − 1 = 9

Answer: 9 edges.

Exercise 2

Why can't an MST contain a cycle?

A spanning tree must be acyclic.

If a cycle exists, one edge from the cycle can be removed without disconnecting the graph, reducing the total cost.

Therefore, an MST can never contain a cycle.

Exercise 3

Which algorithm generally performs better on sparse graphs?

Kruskal's Algorithm is generally preferred for sparse graphs because it efficiently processes a smaller number of edges.

Answer: Kruskal's Algorithm.

Exercise 4

What data structure is commonly used with Kruskal's Algorithm?

Kruskal's Algorithm uses the Disjoint Set Union (DSU) or Union-Find data structure.

It helps detect cycles efficiently while building the MST.

Answer: Union-Find (DSU).

Exercise 5

Calculate the MST cost for the following graph:

A-B = 1
A-C = 4
B-C = 2
B-D = 5
C-D = 3

Select the smallest edges without creating cycles:

  • A-B = 1
  • B-C = 2
  • C-D = 3

Total MST Cost = 1 + 2 + 3 = 6

Answer: MST Cost = 6.
