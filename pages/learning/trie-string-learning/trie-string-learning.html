01

Trie (Prefix Tree)

A Trie is a tree-like data structure whose nodes store the letters of an alphabet. It is exceptionally efficient for tasks like autocomplete, spell checking, and finding strings with a common prefix.

Core Operations

  • Insert: O(L) where L is the length of the word.
  • Search: O(L) to find an exact match.
  • Autocomplete (StartsWith): O(L) to verify if a prefix exists.
Python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word: str) -> None:
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

    def search(self, word: str) -> bool:
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

    def starts_with(self, prefix: str) -> bool:
        node = self.root
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        return True
C++
class TrieNode {
public:
    unordered_map<char, TrieNode*> children;
    bool isEndOfWord = false;
};

class Trie {
private:
    TrieNode* root;
public:
    Trie() { root = new TrieNode(); }
    
    void insert(string word) {
        TrieNode* node = root;
        for (char c : word) {
            if (!node->children.count(c)) node->children[c] = new TrieNode();
            node = node->children[c];
        }
        node->isEndOfWord = true;
    }
};
02

KMP Algorithm (Prefix Function)

Knuth-Morris-Pratt (KMP) searches for a pattern in O(N + M) time. Instead of backtracking the text pointer when a mismatch occurs, it uses an LPS (Longest Prefix Suffix) array to skip characters we know already match.

🧠

The Prefix Function (LPS Array): lps[i] stores the length of the longest proper prefix of the sub-pattern pat[0..i] which is also a suffix of this sub-pattern. Example for "ABABC": [0, 0, 1, 2, 0].

Python
def compute_lps(pat):
    lps = [0] * len(pat)
    length = 0
    i = 1
    while i < len(pat):
        if pat[i] == pat[length]:
            length += 1
            lps[i] = length
            i += 1
        else:
            if length != 0:
                length = lps[length - 1] # Fallback without moving i
            else:
                lps[i] = 0
                i += 1
    return lps
03

Z-Algorithm

The Z-algorithm finds all occurrences of a pattern in a text in linear time. It builds a Z-array where Z[i] is the length of the longest substring starting from S[i] which is also a prefix of S.

💡

Trick: Concatenate the pattern and text like this: Pattern + "$" + Text. Run the Z-algorithm. Any index i where Z[i] == length(Pattern) indicates a match in the text!

04

Rolling Hash (Rabin-Karp)

Rabin-Karp uses a sliding window and a Rolling Hash to match patterns. Instead of comparing characters one by one, it compares the hash values of the current window and the pattern in O(1) time.

Handling Hash Collisions

Because multiple strings can produce the same hash modulo M (a collision), we must handle them:

  • Direct Verification: If hashes match, do a character-by-character comparison (O(L) worst case).
  • Double Hashing: Compute two distinct hashes using different prime bases and modulos. A match is only considered if both hashes match. This drastically reduces collision probability.
05

Aho-Corasick Automaton

When searching for multiple patterns simultaneously, KMP must be run for each pattern. Aho-Corasick builds a Trie of all patterns and adds "failure links", allowing a single pass over the text to find all occurrences of all patterns.

🕸️
Understanding Failure Links (Crucial)

A failure link at a specific node u points to the node v which represents the longest proper suffix of the string formed at u that is also a prefix of some pattern in the Trie.

If you mismatch a character while traversing the Trie, you don't go back to the root. Instead, you follow the failure link to salvage the longest possible matching prefix you've already seen, exactly like the LPS array in KMP, but generalized for a tree.

2. Implement strStr() via KMP

Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Python
def strStr(haystack, needle):
    if not needle: return 0
    lps = compute_lps(needle) # From section 02
    i = j = 0
    while i < len(haystack):
        if haystack[i] == needle[j]:
            i += 1; j += 1
        if j == len(needle):
            return i - j
        elif i < len(haystack) and haystack[i] != needle[j]:
            if j != 0: j = lps[j-1]
            else: i += 1
    return -1
