What is State Compression?

In traditional DP, states are usually indices or remaining capacities. But what if your state depends on a subset of visited cities or assigned tasks? You could pass an array `visited[]`, but arrays can't be used easily as cache keys.

Solution: Because integers are just binary numbers under the hood, we can use the bits of an integer to represent a boolean array! `1010` (binary) means items at indices 1 and 3 are taken, while 0 and 2 are not.

💡

When to use Bitmask DP: Look for very small constraints, typically N ≤ 20. Since a 32-bit integer can store 32 boolean states, 2^20 is roughly 10^6, which fits perfectly within standard time limits.

Clear i-th bit
mask & ~(1 << i)
Toggle i-th bit
mask ^ (1 << i)
All bits 1 (Size N)
(1 << N) - 1

mask & (-mask)

Subset DP: The Assignment Problem

Problem: There are N persons and N tasks. You have a 2D cost matrix where `cost[i][j]` is the cost of assigning person `i` to task `j`. Find the minimum cost to assign exactly one task to each person.

Approach: We use a mask to represent which tasks have already been assigned. The state is just `dp(mask)`, and the number of set bits in the mask tells us which person `i` we are currently assigning.

Python
def min_cost_assignment(cost):
    n = len(cost)
    memo = {}

    def dp(mask):
        # If all tasks are assigned (all N bits are 1)
        if mask == (1 << n) - 1:
            return 0
        
        if mask in memo:
            return memo[mask]

        # The i-th person we are assigning is the number of set bits in mask
        i = bin(mask).count('1')
        ans = float('inf')

        for j in range(n):
            # If task j is NOT assigned yet (j-th bit is 0)
            if (mask & (1 << j)) == 0:
                ans = min(ans, cost[i][j] + dp(mask | (1 << j)))

        memo[mask] = ans
        return ans

    return dp(0)
C++
int memo[1<<20];
int minCostAssignment(int mask, int n, vector<vector<int>>& cost) {
    if (mask == (1 << n) - 1) return 0;
    if (memo[mask] != -1) return memo[mask];

    int i = __builtin_popcount(mask);
    int ans = 1e9;

    for (int j = 0; j < n; j++) {
        if (!(mask & (1 << j))) {
            ans = min(ans, cost[i][j] + minCostAssignment(mask | (1 << j), n, cost));
        }
    }
    return memo[mask] = ans;
}

Traveling Salesman Problem (TSP)

Problem: Given a list of cities and the distances between them, find the shortest possible route that visits every city exactly once and returns to the origin city.

🗺️

The State: dp(mask, u)

Where mask represents the set of cities visited so far, and u is the city we are currently at. We need to keep track of u because the cost to move to the next unvisited city v depends on where we are currently standing.

Python
def tsp(mask, u, dist, n, memo):
    # If all cities have been visited, return to origin (city 0)
    if mask == (1 << n) - 1:
        return dist[u][0]
        
    if memo[mask][u] != -1:
        return memo[mask][u]

    ans = float('inf')
    for v in range(n):
        # If city v is NOT visited
        if (mask & (1 << v)) == 0:
            new_ans = dist[u][v] + tsp(mask | (1 << v), v, dist, n, memo)
            ans = min(ans, new_ans)

    memo[mask][u] = ans
    return ans

# Initial call: tsp(1, 0, dist, n, memo) -> mask=1 means city 0 is visited
C++
int memo[1<<15][15];
int tsp(int mask, int u, int n, vector<vector<int>>& dist) {
    if (mask == (1 << n) - 1) return dist[u][0];
    if (memo[mask][u] != -1) return memo[mask][u];

    int ans = 1e9;
    for (int v = 0; v < n; v++) {
        if (!(mask & (1 << v))) {
            ans = min(ans, dist[u][v] + tsp(mask | (1 << v), v, n, dist));
        }
    }
    return memo[mask][u] = ans;
}

1. Determine if a number is a power of 2

Write an O(1) expression to check if a positive integer N is a power of 2 using bitwise operations.

Hint: A power of 2 has exactly one bit set. Subtracting 1 from it flips all the bits to the right of the set bit (including the set bit itself).

def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0
