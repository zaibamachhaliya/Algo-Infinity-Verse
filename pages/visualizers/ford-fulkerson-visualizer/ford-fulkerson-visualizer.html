Ford-Fulkerson Algorithm Concepts

Augmenting Path Search Strategies

The **Ford-Fulkerson** method works by searching for any path from source $S$ to sink $T$ in the residual graph with remaining capacity. The strategy used to find this path affects the complexity:

- **Edmonds-Karp (BFS)**: Guarantees finding the path with the fewest edges. This bounds the total augmentations and yields a polynomial runtime of $O(V E^2)$. - **DFS Pathfinding**: Searches deep. In networks with very large capacities, poor path choices can cause DFS to take $O(f \cdot E)$ augmentations, where $f$ is the max flow.

Bottlenecks & Max-Flow Min-Cut

When an augmenting path is discovered, we identify the edge with the **minimum remaining capacity** along that path. This value is the **bottleneck**.

We can push exactly this bottleneck amount of flow through the path. By the **Max-Flow Min-Cut Theorem**, the value of the maximum flow is equal to the capacity of the minimum cut separating source $S$ from sink $T$. When no more augmenting paths can be found, the network has reached maximum flow.
