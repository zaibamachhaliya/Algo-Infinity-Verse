🌐 Distributed Data Structures

Skip Graph Visualizer

No head. No root. Every node is equal. Search starts anywhere.

Skip Lists need a single head node — kill it and the whole structure is gone. Skip Graphs distribute the same multilevel structure across every node using membership vectors. Any node can start a search, and the failure of any node only affects paths that use it. The P2P DHT structure that BitTorrent and distributed databases actually rely on.

Skip List vs Skip Graph — Key Differences
Skip List
Single head node — required entry point
Kill the head → entire structure unreachable
Simple to implement, excellent for single-machine
Centralized — not suitable for distributed systems
O(log n) search, insert, delete
No coordination between nodes needed
Skip Graph
Every node is an equal entry point
Node failures only affect paths using that node
More complex — each node stores multiple neighbor lists
Fully decentralized — ideal for P2P DHT
O(log n) search with high probability
Membership vector coordination needed on join
