Anytime Guarantees — Not All Algorithms Promise the Same Thing
Weighted A*
✅ Monotonic

Each time the weight ε decreases, the new solution is provably no worse. Quality curve never goes up. Strongest anytime guarantee of the three.

Simulated Annealing
⚠️ Best-tracked only

The CURRENT solution can get worse at any step (that's the point — escaping local optima). Only the separately tracked "best seen so far" is monotonic.

Genetic Algorithm
🟡 Elitism-dependent

With elitism (best individual always survives), best-of-generation is monotonic. Without it, even the best individual can be lost between generations.
