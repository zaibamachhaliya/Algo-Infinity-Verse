📖 How KMP Uses the Prefix Function

01

Compute LPS

For each pattern position, the prefix function stores the longest proper prefix that is also a suffix. That value is the jump target after a mismatch.

02

Match Without Backtracking

When text and pattern mismatch, KMP does not rewind the text pointer. It falls back inside the pattern using the LPS array.

03

Why It Is Fast

The algorithm never rechecks text characters unnecessarily. The total runtime stays linear in the text plus pattern length.
