Execution Trace

1. h(S) = |V|, h(others) = 0
2. Push full capacity from S
3. While active vertex u exists:
4. Find admissible edge (u, v)
5. If found: Push(u, v)
6. Else: Relabel(u)
7. Return Flow entering Sink

Theory

Push-Relabel operates locally by maintaining a preflow. Nodes can hold "excess" flow. Flow is strictly pushed downhill ($h(u) > h(v)$).

  • Push Moves flow downhill using residual capacity.
  • Relabel Raises a node's height to create a downhill slope.

Complexity

Time (Worst Case) O(V²E)
Space O(V²)

Faster than Edmonds-Karp for dense graphs due to localized operations rather than full BFS path searches.
