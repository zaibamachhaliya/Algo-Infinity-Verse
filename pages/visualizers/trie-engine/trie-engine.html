Theory

A Trie (Prefix Tree) is an N-ary tree used for efficient string matching. Paths are shared between words with common prefixes to save space.

  • Root Empty starting node.
  • Edge Represents a character.
  • End Node Marks a valid dictionary word termination.

Complexity

Time (Insert/Search) O(L)
Space O(N × L)

L is the length of the word, and N is the number of words. Lookups are independent of the total dictionary size.
