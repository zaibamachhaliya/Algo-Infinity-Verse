Static (Backward-Taken) —

Always predicts "taken" for backward branches (loops), "not taken" for forward branches (ifs). No history tracking — the simplest possible strategy.

1-Bit Predictor —

Remembers only the last outcome and predicts the same again. Fast to adapt, but flips on every single mispredict — punished hard by alternating patterns.

Hits: 0 Misses: 0 Stall cycles: 0
2-Bit Saturating Counter —

The classic — used in real CPUs since the 1990s. Needs two consecutive wrong guesses before flipping its prediction, tolerating one-off anomalies.

Hits: 0 Misses: 0 Stall cycles: 0
Two-Level Adaptive —

Tracks the last N outcomes as a "history pattern", then uses a 2-bit counter per pattern in a Pattern History Table. Learns repeating cycles that simpler predictors can't see.
