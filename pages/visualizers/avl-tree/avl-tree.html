5. If BF > 1 (Left Heavy):
If val < left.val: Right Rotate (LL)
Else: Left-Right Rotate (LR)
6. If BF < -1 (Right Heavy):
If val > right.val: Left Rotate (RR)
Else: Right-Left Rotate (RL)

Theory

An AVL Tree is a self-balancing Binary Search Tree. It strictly ensures that the height difference (Balance Factor) between the left and right subtrees of ANY node is never more than 1.

  • BF ∈ {-1, 0, 1} Subtree is balanced.
  • BF > 1 or < -1 Subtree is unbalanced. A mathematical rotation is instantly triggered to restore $O(\log N)$ height.
