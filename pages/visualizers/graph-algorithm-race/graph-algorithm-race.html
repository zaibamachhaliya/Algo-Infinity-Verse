🎓 Why They Differ

BFS

Breadth-First Search

Explores level by level. Guarantees shortest path in unweighted graphs. Ignores edge weights — treats all edges as cost 1. Wide and shallow.

O(V+E)Unweighted optimal
DFS

Depth-First Search

Goes as deep as possible before backtracking. No path optimality guarantee. Memory efficient on tall graphs. Often finds a path fast but not the shortest.

O(V+E)No optimality
Dijkstra

Dijkstra's Algorithm

Explores by minimum cumulative cost using a priority queue. Optimal for weighted graphs with non-negative edges. Explores more nodes than A* but finds optimal path.

O((V+E) log V)Weighted optimal
A*

A* Search

Dijkstra + heuristic. Uses f(n) = g(n) + h(n) to guide search toward target. Visits fewer nodes than Dijkstra when heuristic is good. Optimal if heuristic is admissible.

O((V+E) log V)Heuristic-guided
