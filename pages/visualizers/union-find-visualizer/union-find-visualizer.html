📖 How Union-Find Works

01

Make Set — O(1)

Each element starts as its own set. parent[x] = x, rank[x] = 0. There are n disjoint sets initially.

Time: O(1)n components
02

Find with Path Compression — O(α(n))

Find the root of x's set. Path compression makes every node on the path point directly to the root — flattening the tree.

Time: O(α(n)) amortizedFlattens tree
03

Union by Rank — O(α(n))

Attach the shorter tree under the taller one. The rank is an upper bound on tree height. This keeps trees shallow.

Time: O(α(n)) amortizedKeeps trees flat
